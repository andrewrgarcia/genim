# Image generator with speech command

# install library
## on mac
```
pip install -r requirements-mac.txt
```
## on windows 
```
pip install -r requirements-win.txt
```

# run gui
```
python gui.py
```
## with flask run (no GUI)
```
export FLASK_APP=main
export FLASK_ENV=development
flask run -p 1234
```
